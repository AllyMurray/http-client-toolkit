# HTTP Client Toolkit

A modular HTTP client toolkit with pluggable caching, deduplication, and rate limiting.

## Packages

| Package | Description |
|---------|-------------|
| `@http-client-toolkit/core` | Core HTTP client with store interfaces |
| `@http-client-toolkit/store-memory` | In-memory store implementations |
| `@http-client-toolkit/store-sqlite` | SQLite store implementations |
